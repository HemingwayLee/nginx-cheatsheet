# nginx-cheatsheet

* Use nginx in any linux distribution
```
nginx 
nginx -s stop
nginx -s reload
```

## Ubuntu
* config file path:
```
/etc/nginx/sites-enabled/default
```

* default working folder
```
/var/www/html
```

## Centos
* config file path:
```
/etc/nginx/nginx.conf
```

* default working folder
```
/usr/share/nginx/html
```
